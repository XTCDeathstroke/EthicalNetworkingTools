# Contributing

Thanks for contributing to ENWTS.

## Guidelines

- Keep changes focused and well documented.
- Prefer safe defaults and clear warnings.
- Do not add features intended for stealth, evasion, or unauthorized access.

## Development

- Python 3.11+
- Run locally:

```bash
python ENWTS.py
```

## Pull requests

- Describe what you changed and why.
- Include screenshots for UI changes when possible.
